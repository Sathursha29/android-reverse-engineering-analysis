# Android Reverse Engineering & Malware Simulation

## Overview

This project was completed as part of the IE3112 Mobile Security module at SLIIT.

The objective was to reverse engineer an Android application, analyze its architecture, inject controlled Trojan-like functionality for educational purposes, and evaluate security risks and mitigation strategies.

> This project was conducted strictly for academic and educational purposes in a controlled environment.

---

## Objectives

- Reverse engineer an Android APK
- Analyze Smali bytecode
- Understand Android application architecture
- Simulate controlled malware behavior
- Evaluate security risks
- Recommend mitigation strategies
